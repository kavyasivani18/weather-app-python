#  Weather App — Python

A command-line weather application that fetches **real-time weather data** 
using the OpenWeatherMap API.

##  Features
-  Search weather by any city name
-  Displays temperature (°C), humidity, and weather condition
-  Real-time data using live API calls
-  Clean error handling for invalid cities

##  Tech Stack
- **Language:** Python 3
- **Library:** Requests
- **API:** OpenWeatherMap API

##  How to Run

### 1. Clone the repository
git clone https://github.com/kavyasivani18/weather-app-python.git
cd weather-app-python

### 2. Install dependencies
pip3 install requests

### 3. Add your API key
Open weather.py and replace:
API_KEY = "your_api_key_here"
Get your free key at: https://openweathermap.org/api

### 4. Run the app
python3 weather.py
